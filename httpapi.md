# 通用约定
HttpApi仅支持post方法，报文内容采用JSON方式封装。数据编码统一采用UTF-8。

## 数据包定义
接口地址形如http://127.0.0.1:18080/QryLastTicker
消息格式为json字符串
```json
{
    "param1":"参数1",
    "param2":"参数2",
    ……
}
```
## 合约关键字命名规则
证券分类简称_商品代码_合约代码（期数）

例如：FO_CL_1811表示-国际原油1811

如果没有合约代码（期数）表示主力合约

例如：FI_sc表示国内原油主力合约

非期货商品不含期数

例如：D_BTC表示数字货币比特币

证券分类简称表：

| 中文名称 | 简称 |
| -------- | ---- |
| 国际期货 | FO   |
| 国内期货 | FI   |
| 外汇     | C    |
| A股      | SA   |
| 港股     | SHK  |
| 美股     | SUS  |
| 数字货币 | D    |
| 维胜品种 | V    |
